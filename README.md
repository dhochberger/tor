# Travel Order

### Requirements:
- Python

To install the dependencies:
```
$ make install
```

To test the project:
```
$ make test
```


To launch the project:
```
$ make dev
```

To launch the project in daemon mode:
```
$ make prod
```

To shutDown daemon mode:
```
$ make stop port=[port]
```

To upgrade pip:
```
$ python3 -m pip install --upgrade pip
```
